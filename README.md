<b>Link shortener app</b><br>
It uses Golang, Fiber, Gorm.

<b>Functional:</b><br>
- Make short link
- Get original link by short equivalent


<b>Swagger:</b><br>
`/swagger/index.html`


<b>Commands:</b><br>
- `docker compose up`<br>
for run server<br>
- `swag init -g ./cmd/main.go -o ./docs`<br>
for generate swagger<br>
